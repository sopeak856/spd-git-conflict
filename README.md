# SPD Week 4 Lab Demo - Advanced Git & Branching

This is the official demo repository for Week 4 lab: **Feature Branching and Merge Conflict Resolution**.

## Project Description
A simple "Software Project Development" portal website used to practice:
- Feature branching
- Creating merge conflicts
- Resolving conflicts (basic + advanced)

## Files for Practice
- `index.html` → Basic conflict (header changes)
- `about.html` → Modify/Delete conflict
- `assets/logo.png` → Binary file conflict
- `old-page.html` (will be created during lab) → Rename + Modify conflict

## How to Use
1. Clone this repo
2. Follow the **Student Worksheet** Step-by-Step
3. Open `index.html` in browser to see the live demo

Happy Git practicing!
